Testing your Integration

To test your integration of UsherJS, be sure to configure Usher to use Staging and then use one of the Test Campaigns over at the Usher Staging Environment!

const usher = Usher({ apiUrl: "https://app.staging.usher.so/api" });

A Campaign has already been configured with a Destination URL of http://localhost:3001/

This means that when this Invite Link is used, a redirect to http://localhost:3001/ will take place.

To get started on this Staging Campaign, connect a wallet and Start a Partnership!

All Partnerships submitted to this Staging Environment are managed on the Ceramic Testnet.
